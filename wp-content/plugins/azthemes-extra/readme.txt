=== sunio Extra ===
Contributors: sunio
Tags: widgets, meta box, metaboxes, metabox, sunio

== Description ==

sunio Extra add extra features to [sunio](https://sunio.net/) like widgets, metaboxes, activate/deactivate the customizer sections, enable/disable the theme's scripts and styles
This plugin requires the [sunio](https://sunio.net/) theme to be installed.

== Installation ==

1. Upload `sunio-extra` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!

== Frequently Asked Questions ==

= I installed the plugin but it does not work =

This plugin will only work with the [sunio](https://sunio.net/) theme.
